# dotfiles bootstrap

Configuration files and bootstrap manager build thanks to the Internet.

## Usage

	git clone git@github.com:LEI/dotfiles.git ~/.dotfiles && cd ~/.dotfiles
	~/.dotfiles/bootstrap install

	dot --help
